# Report Format

Use this shape for the final output.

## 1. Product Target

- product name
- brand
- variant or size if known
- matching note explaining how listings were mapped

## 2. Coverage Summary

Provide a compact table or bullet list with:

- channel
- source count
- evidence quality
- notes

## 3. Top Complaint Themes

For each theme include:

- theme name
- why it matters
- confidence
- estimated spread across channels

## 4. Representative Evidence

For each theme, include 1-3 examples:

- short paraphrase or safe quote
- source link
- channel
- note about context

## 5. Cross-Channel Patterns

Answer questions like:

- Which complaints show up in more than one channel?
- Which complaints are channel-specific?
- Are retailer reviews and community discussion telling the same story?

## 6. Confidence And Gaps

State:

- missing channels
- weak product matching
- low sample size
- inaccessible pages
- likely bias

## 7. Suggested Next Step

Recommend only one next step, for example:

- verify a suspected safety complaint with more retailer pages
- compare against a competitor product
- expand to customer-service and return-reason data
