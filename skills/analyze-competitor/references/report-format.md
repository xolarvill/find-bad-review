# Report Format

Use this shape for the final output.

## 1. Focal Target

- focal brand or product
- category
- geography or channel scope if relevant
- matching note explaining what exactly was compared

## 2. Competitor Set

List each included competitor with a short inclusion note.

Preferred shape:

```md
- Competitor: Ruffwear
  Inclusion logic: premium direct substitute with strong outdoor-performance positioning
```

## 3. Coverage Summary

Provide a compact table or bullet list with:

- competitor
- sources used
- evidence quality
- notes

## 4. Comparison Table

Normalize the key fields being compared. Default columns often include:

- competitor
- role
- price or price band
- focal offer
- main claims
- trust proof
- notable gap or weakness

Adjust columns to the user's question. Do not add decorative columns that do not affect the decision.

## 5. Key Differences

For each meaningful difference include:

- difference statement
- why it matters
- which competitors it separates
- confidence
- evidence attached directly below

Preferred shape:

```md
### 1. Difference statement

- Why it matters:
- Confidence:
- Seen in:

Example pages:
- [Page title](url) - short note
- [Page title](url) - short note
```

## 6. Commercial Judgments

This section is required when the evidence supports business-level conclusions.

Preferred shape:

```md
- Judgment: The focal offer is not obviously overpriced, but it is under-proven relative to competitors.
  Basis: Competitors surface review counts, guarantees, and clearer materials proof earlier.
  Why it matters: The likely gap is trust density, not headline price.
```

Every judgment must be tied to the comparison evidence.

## 7. Confidence And Gaps

State:

- missing competitors
- weak SKU matching
- inconsistent pricing context
- inaccessible pages
- channels with low evidence quality

## 8. Suggested Next Move

Recommend one next step only, for example:

- run a deeper bad-review pass on the strongest competitor
- compare above-the-fold conversion messaging only
- expand to retailer and Amazon listing comparison
